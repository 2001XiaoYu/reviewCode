# 记账项目 #
    本项目实现了简单的注册、登录（退出登录）、个人记账、添加账单、删除账单、更新账单、获取账单列表等功能。
    技术栈：node + express + ejs + JWT, 数据库：MongoDB 接口工具: Apifox
